WHY VIBECHECK

Why Use VibeCheck?

Understanding audience opinion should not require reading thousands of comments.

AI Powered Sentiment

A LinearSVC model trained on 100K+ YouTube comments classifies each comment as Positive, Neutral, or Negative with high accuracy.

YouTube API Integration

Real-time comment extraction using the YouTube Data API v3. Supports pagination for deep analysis.

FastAPI Backend

Lightning-fast Python backend with CORS support, batch prediction endpoints, and zero cold-start delay.

Interactive Analytics

Doughnut charts, animated progress bars, timeline graphs, keyword clouds, and comment explorers.

ML Pipeline

Full end-to-end pipeline: preprocessing, TF-IDF feature engineering, model inference, and post-processing.

Export Reports

Download a full PDF report or export raw predictions as CSV for further analysis.

01

User Input

A YouTube video URL (standard, short, or Shorts format) is validated and the video ID is extracted client-side.

02

API Calls

The YouTube Data API v3 fetches video metadata and top-level comments with author data and like counts.

03

Batch Processing

Comment texts are batched and sent to the FastAPI /predict/batch endpoint in a single POST request.

04

Inference Pipeline

Texts are translated (Gemini API), vectorized with TF-IDF, and classified by a trained LinearSVC model.
